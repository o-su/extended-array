# Extended Array

Extends native array with additional methods:

-   get
-   set
-   remove
-   delete
-   isEmpty
-   isNotEmpty
-   clear
-   clone
-   sum
-   min
-   max

## Installation

```bash
npm install --save extended-array
```

## Example Usage

```typescript
import "extended-array";

const myArray: string[] = [];

myArray.set(0, "firstElement");
myArray.get(0);
myArray.clear();
```
